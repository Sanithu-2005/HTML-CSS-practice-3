Weekly Reflection

What I Learned

This week was a deep dive into HTML Forms and CSS Float Layouts. I learned how to use fieldset and legend to group related data logically, which significantly improves accessibility. I also explored various input types like tel and email, which provide built-in validation on mobile devices.

Challenges Faced

The biggest challenge was mastering the float property. I initially struggled with elements collapsing into each other until I implemented the clearfix hack and the clear: both property. Understanding that floated elements are taken out of the normal document flow was a "lightbulb" moment for me. I also had to be careful with box-sizing: border-box to ensure my 48% width containers didn't break to a new line when padding was added.
